O desenvolvimento dos computadores

(voltar ao início)

A evolução da tecnologia é constante. Para o início, precisamos traçar uma cronologia da evolução dos computadores. Devemos destacar também que a Internet teve uma longa trajetória dde evolução, paralela à das máquinas, embora não seja tratado sobre isso neste artigo. Dividimos o desenvolvimento dos computadores em gerações, podendo seguramente afirmar que já estamos na 4ª, embora há quem diga que já chegamos na 5ª geração. Portanto, vamos clarear um pouco mais sobre essa questão:

1ª geração: Os computadores de primeira geração, datados da década de 40 eram simplesmente estruturas gigantescas para a realização de cálculos complexos. No auge da II Guerra Mundial, era muito importante que o desenvolvimento da tecnologia estivesse presente. Para isso, temos o Mark I (1944), desenvolvido em Harvard. Com 5 toneladas, este gigante eletromecânico foi considerado o primeiro computador. Nesta mesma geração encontramos o Colossus (1944) da Alemanha, capaz de realizar criptoanálises voltadas para a guerra. Por fim, podemos citar o ENIAC (1946) também norte-americano, que realizava cálculos de balísticas.

2ª geração: Entre 1959 e 1964 encontramos a segunda geração de computadores. A grande evolução aqui presente é a mudança de válvulas para transitores, o que contribuiu na diminuição do tamanho das máquinas.

3ª geração: A terceira geração de computadores é datada entre 1965 e 1970. É nesse período em que os transitores são substituidos por chips, reduzinddo ainda mais o tamanho das máquinas.

4ª geração: A quarta geração é considerada entre 1971 e os dias atuais. É neste período no qual a tecnologia na forma de computadores se popularizou e atingiu bilhões de pessoas. Os microprocessadores permitiram a existência dos Personal Computers, familiarmente conhecidos como PCs, devido ao pequeno tamanho ocupado no hardware.

5ª geração: Será que existe mesmo? Algumas pessoas acreditam que o desenvolvimento dos computadores já nos permitiu chegar a 5ª geração, mas isso não é unanimidade. Contudo, todos os atuais processos que vivemos até mesmo no dia-a-dia da humanidade, pode nos fazer refletir sobre a existência da 5ª geração. Temas como computadores quânticos, inteligência artificial, metaverso e criptomoedas, que caracterizariam esta geração estão cada vez mais em discussão.

Clients? Servers?

(voltar ao início)

Clients e servers são apenas lados no qual a informação percorre, sendo requisitada no processamento. Imagine a seguinte situação:

Você está fazendo compras presencialmente. Você entra na loja, escolhe o produto, paga e espera o pacote ser entregue.

Vamos utilizar essa analogia para entender a diferença entre client e server! O client é o equivalente a você mesmo, entrando na loja e fazendo suas compras. O local onde está o seu pacote, esperando para ser entregue, é o server. É nesse local onde há diversos boxes com outros pacotes, que cujos donos estão aguardando para receber. Trazendo para a realidade computacional, client é qualquer dispositivo que consome o conteúdo via Internet - o meio de comunicação entre client e server. E server é o local onde as informações (conteúdo) ficam armazenadas! Quando um client requisita uma informação através de um navegador, este retorna arquivos - que podem ser estáticos - e permite que o client receba essas informações.

Representação do modelo Cliente-Servidor. Fonte: Wikipedia, 2023.

Existem diversos tipos de servidor. Podemos caracterizar alguns:

  • Arquivos: um tipo de servidor que centraliza todos os arquivos de uma empresa, por exemplo.
  • Segurança: servidor que realiza bloqueios de segurança contra acessos externos maliciosos.
  • Streaming: servidor no qual há a concentração de dados que são transmitidos on demand para o client.
  • E-mail: servidor exclusivo para o envio e recebimento de correio eletrônico.
  • Web: servidor de dados voltado para o uso da Internet.
  • Proxy: um servidor anterior ao servidor de dados ou web, que cria regras para o uso de internet de uma determinada rede. Também pode ser usado como autenticador, com uso de usuário e senha.
  • Intranet: uma espécie de servidor web sem a internet, para uso em determinada rede local.
  • Data Center: um conjunto de servidores, quando há a necessidade de um grande volume de dados a serem armazenados.

Tipos de linguagens de programação

(voltar ao início)

Linguagem de programação é uma linguagem escrita, que com sua sintaxe e sua semântica cria um conjunto de instruções e regras para gerar programas, também conhecidos como softwares. Existem linguagens que são server-side e outras que são client-side. Cada linguagem tem a sua característica.

Linguagens server-side são linguagens que trabalham no processamento de informações dentro do servidor. São exemplos de linguagem server-side: PHP, C#, Java, Ruby, JavaScript. Linguagens client-ide são linguagens que processam a informação para o lado do cliente através de uma aplicação web (soluções que possuem a internet como meio de comunicação e que não precisam ser instaláveis em uma máquina). Um exemplo de uma linguagem client-side é o JavaScript*. E, embora este seja um curso voltado para o HTML, é importante lembrar que HTML não é uma linguagem de programação! O HTML é um arquivo de marcação, que trabalha com textos delimitados por nomes (tags) que o navegador consegue interpretar.

A linguagem de programação deve ser considerada acima de tudo, um meio para a resolução de problemas.

*O JavaScript é uma linguagem tanto server quanto client-side e a forma como essa linguagem funcionará depende de mecanismos de leitura, como o V8.
